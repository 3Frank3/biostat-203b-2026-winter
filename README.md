# biostat-203b-2026-winter
Machine Learning on MIMIC-IV Data: ICU Ventilation Duration Analysis

•	Analyzed 500k+ ICU clinical data to identify patterns associated with prolonged invasive ventilation, supporting data-driven insights into early clinical decision-making and patient care.
•	Built a reproducible end-to-end data pipeline in R and Linux to extract, clean, and integrate patient demographics, procedures, and time-stamped ICU events into an analytics-ready dataset for modeling.
•	Applied feature engineering across 30+ clinical variables and trained machine learning models to classify ventilation duration, achieving ROC AUC ~0.80 with Brier score ~0.17. 
•	Conducted exploratory data analysis and visualization through ggplot2 to identify key factors associated with prolonged ventilation and communicate findings through analytical outputs.
